## Ambiente
- Linux Mint 19 Tara
- Python 3.6
- Mysql

##### Ações necessárias:
---
> Executar no terminal:
* sudo apt install python3-pip 
* pip3 install --upgrade setuptools
* pip3 install mysql-connector 
* mysql -u usuario_mysql -psenha < query.sql (Na raiz do projeto)
